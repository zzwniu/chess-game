# 🀄 中国象棋 (Chinese Chess)

HTML5 中国象棋，支持人机对战和双人对战。纯前端实现，打开即玩！

## 🎯 功能特色

- 🀄 **完整规则** — 所有棋子走法：将/帅、士/仕、象/相、馬、車、炮/砲、兵/卒
- 🏰 **九宫限制** — 将帅不出九宫，士仕斜走九宫内
- 🐘 **塞象眼** — 相/象走田字，蹩脚不可走
- 🐴 **蹩马脚** — 馬走日字，蹩脚不可走
- 💣 **炮翻山** — 炮直线移动，隔一子吃子
- ⚔️ **将帅对面** — 飞将检测
- 🤖 **AI 对手** — 4个难度等级，Minimax + Alpha-Beta 剪枝
- 👥 **双人模式** — 两人轮流在同一台电脑上对弈
- ↩️ **悔棋** — 支持 `Ctrl+Z`

## 🎮 难度等级

| 等级 | 搜索深度 | 说明 |
|------|---------|------|
| 🟢 初级 | 1层 | 只看一步，适合新手 |
| 🟡 中级 | 2层 | 向前看两步，有一定棋力 |
| 🟠 高级 | 3层 | 向前看三步，棋力较强 |
| 🔴 大师 | 4层 | 向前看四步，棋力强劲 |

## 🕹️ 操作方式

| 操作 | 方式 |
|------|------|
| 选择棋子 | 鼠标点击 |
| 走棋 | 点击目标交叉点 |
| 新游戏 | 点击按钮 或 按 `N` |
| 悔棋 | 点击按钮 或 `Ctrl + Z` |

## 🚀 快速开始

用浏览器打开 `index.html` 即可开始！

```bash
git clone https://github.com/zzwniu/chess-game.git
cd chess-game
# 双击 index.html
```
