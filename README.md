# ♟️ 国际象棋 (Chess)

HTML5 国际象棋，纯前端实现，支持完整的国际象棋规则。打开即玩！

## 🎯 功能特色

- ✅ **完整规则** — 所有棋子走法、王车易位、吃过路兵、兵升变
- ✅ **将军/将杀检测** — 自动检测将军、将杀、逼和
- ✅ **合法走子高亮** — 点击棋子显示所有合法走法
- ✅ **吃子记录** — 侧边栏显示双方被吃棋子
- ✅ **走棋历史** — 完整走棋记录，支持代数记谱
- ✅ **悔棋功能** — 一步悔棋，支持 `Ctrl+Z` 快捷键
- ✅ **精美UI** — 渐变棋子、阴影效果、坐标标注
- ✅ **升变弹窗** — 兵到底线弹出升变选择

## 🎮 操作方式

| 操作 | 方式 |
|------|------|
| 选择棋子 | 鼠标点击 |
| 走棋 | 点击合法目标格子 |
| 升变 | 在弹出的窗口中选择棋子 |
| 悔棋 | 点击按钮 或 `Ctrl + Z` |
| 新游戏 | 点击按钮 |

## ⌨️ 快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl + Z` | 悔棋 |

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/zzwniu/chess-game.git
```

2. 用浏览器打开 `index.html` 即可开始对弈！
